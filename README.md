# Indian Elections Dashboard

This project is a Power BI dashboard created to explore and understand Indian election data in a simple and interactive way. It focuses on presenting key information clearly so that patterns and trends can be easily observed.

## Overview

The dashboard helps in analyzing election results, party performance, and constituency-level competition. Instead of going through raw data, users can quickly gain insights through visualizations.

## Features

* Summary cards showing overall election statistics
* Visualization of seats won by different political parties
* Classification of results based on winning margins
* Analysis of re-election trends
* Filters to explore data by year, state, party, and constituency

## Tech Stack

* Power BI
* Dataset in CSV/Excel format

1. Clone or download the repository
2. Open the Power BI file (.pbix) using Power BI Desktop
3. Use the filters and visuals to explore the data

This project was built as part of a data analytics learning process. It shows how raw data can be converted into meaningful insights using visualization tools.
